记账功能# 个人记账软件 - GDUF爪哇部后端组第一轮考核
基于C++实现的轻量级个人记账软件，支持用户管理、记账、统计分析、数据持久化等核心功能，满足个人日常收支管理需求。

## 功能特性
### 1. 用户管理
- ✅ 用户注册/登录：支持多用户账号创建与身份验证
- ✅ 用户数据隔离：不同用户的记账数据完全独立，互不干扰
- ✅ 管理员功能：
  - 管理预设消费类型（增/删）
  - 重置普通用户密码

### 2. 核心记账功能
- ✅ 收支记录：支持收入/支出记录录入
- ✅ 增删查操作：对收支记录进行新增、删除、查询
- ✅ 全量查看：支持查看当前用户所有收支记录
- ✅ 预设消费类型：用户记账时选择预设类型，无需手动输入

### 3. 数据统计分析
- ✅ 总余额统计：自动计算当前用户总收入-总支出的余额
- ✅ 按类型统计：统计各消费类型的收支金额
- ✅ 消费频次统计：统计各消费类型的使用次数

### 4. 数据持久化
- ✅ 所有用户信息、收支记录、消费类型数据均支持本地持久化存储，重启程序后数据不丢失

## 技术栈
- 开发语言：C++（标准C++11及以上）
- 数据存储：本地文件存储（文本文件/二进制文件）
- 编译环境：支持Windows/Linux/macOS（GCC/Clang/MSVC）

## 快速开始
### 编译运行
```bash
# 克隆仓库
git clone https://github.com/你的用户名/accounting-software.git
cd accounting-software

# 编译（以GCC为例）
g++ -std=c++11 main.cpp -o accounting_app

# 运行
./accounting_app  # Linux/macOS
# 或
accounting_app.exe  # Windows
```

### 基础使用流程
1. 启动程序后，选择「注册」创建普通用户账号，或使用管理员账号登录（默认管理员账号可查看代码配置）
2. 登录后进入主菜单，可选择：
   - 「记账」：录入收入/支出记录，选择预设消费类型
   - 「查看记录」：浏览所有收支明细
   - 「统计分析」：查看余额、类型收支、消费频次
   - 「修改记录」：删除/编辑已有收支记录
3. 管理员登录后额外提供「管理消费类型」「重置用户密码」功能

## 目录结构
```
accounting-software/
├── src/                # 源代码目录
│   ├── user/           # 用户管理模块（注册、登录、权限）
│   ├── record/         # 收支记录模块（增删查）
│   ├── statistics/     # 统计分析模块
│   ├── type/           # 消费类型管理模块
│   ├── persistence/    # 数据持久化模块
│   └── main.cpp        # 程序入口
├── data/               # 数据存储目录（自动生成）
│   ├── users.dat       # 用户数据文件
│   ├── records.dat     # 收支记录文件
│   └── types.dat       # 消费类型文件
└── README.md           # 项目说明
```

## 核心模块说明
| 模块          | 功能说明                                  |
|---------------|-------------------------------------------|
| 用户管理      | 处理用户注册、登录验证、权限控制          |

| 数据持久化    | 将内存数据写入本地文件，启动时加载数据    |
